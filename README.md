5. Онлайн-библиотека / каталог книг
Описание:
Сервис, где можно добавлять книги, авторов, жанры, искать по фильтрам.

Что качает:
Сущности: Book, Author, Genre
Поиск: по названию, автору, жанру
Пагинация/сортировка (хотя бы примитивная)
добавить книгу с автором и жанром
вывести книги по:
  автору
  жанру
  поиску по названию

API для добавления автора/жанра
API для топ-книг (если добавить поле рейтинг)

Логика:
Автор не может быть NULL
Название книги уникально в системе
Поиск регистронезависимый
